Flowerify is a web application that enables a user to input an image of a flower. The proposed system will upload, process and analyse the image and output the name and general care information for the identified flower with reference to a known database. The preprocessing and classification task will be undertaken using python Keras. The proposed app will also aid in the maintenance and upkeep of the flower such as when and where to plant this flower; possible pests and diseases. The aim of the app ‘Flowerify’ is to help users to detect, recognise and understand flowers. The author plans to accomplish this by providing a platform for users to upload, process and analyse the flower image. The recommender system will also be implemented in this application for recommending similar flowers to the user input flower. For Flowerify to be an intuitive web app beneficial to users from all backgrounds, the further objectives of the project are to create a visually appealing and effective user interface.
